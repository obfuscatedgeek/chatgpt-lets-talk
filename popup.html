ChatGPT - Let's talk

Adds keyboard shortcuts to ChatGPT site which allows you to talk to and hear AI speak in the site.

You can now change the voice (language feature is totally experimental)

Currently 3 actions are available, press the combination of keys mentioned below to perform the respective action.

Change voices:

Voices : language (select a voice to hear it) Pitch Rate

Keyboard shortcuts:

Action Windows macOS
Talk :   The extension is ready & actively listening, ask your question using your voice / microphone. Alt+⇧Shift+F ⌥ Option+⇧Shift+F
Hear :  Although the extension will read the answer out loud once it's finished generating it, by pressing these keys, the last answer will be read again. Alt+⇧Shift+J ⌥ Option+⇧Shift+J
Stop :  If pressed while the answer is being read, the extension will stop reading. Alt+⇧ Shift+P ⌥ Option+⇧Shift+P
